# AM Allocation Tool

This repository contains two versions of the allocation app.

## Versions

- app.py  
Production-stable version (integer AM only)

- app_v2.py  
Decimals version (0.1 AM steps)

## Run

Install dependencies:

pip install -r requirements.txt

Run stable version:

streamlit run app.py

Run decimals version:

streamlit run app_v2.py
